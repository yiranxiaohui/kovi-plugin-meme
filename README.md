# kovi-plugin-meme 🎭

一个基于 **Kovi** 的趣味表情 / Meme 插件  
支持多种互动指令，让你的 Bot 更有“灵魂”。

---

## ✨ 功能介绍

`kovi-plugin-meme` 提供了一组轻松有趣的互动指令，主要用于群聊或私聊中的娱乐互动。

支持 **@指定用户** 触发，对方会收到对应的 Meme / 行为反馈。

---

## 📦 安装方式

使用 `cargo kovi` 进行安装：

```bash
cargo kovi add kovi-plugin-meme
````

安装完成后，按 Kovi 插件规范启用即可。

---

## 🚀 使用方式

以下指令均支持 **@用户** 形式触发：

| 指令                | 说明        |
| ----------------- | --------- |
| `摸摸头@用户`          | 轻轻摸摸对方的头  |
| `捶爆@用户` / `爆捶@用户` | 对目标进行强力输出 |
| `戒导@用户`           | 对目标进行“戒导” |
| `小丑面具@用户`         | 给对方戴上小丑面具 |
| `小丑@用户`           | 直接小丑化     |
| `手枪@用户`           | 🔫        |
| `上香@用户`           | 🕯️       |
| `催眠app@用户`        | 进入催眠状态    |

> ⚠️ 指令文本需完整匹配，且必须包含 @用户

---

## 🧩 示例

```
摸摸头@小明
捶爆@张三
小丑面具@李四
```
